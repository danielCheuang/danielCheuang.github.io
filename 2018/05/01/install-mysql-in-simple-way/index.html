linux系统简单方式安装mysql

deepind linux系统

1.apt-get install mysql-server
中途会要求设置mysql root用户密码

2.apt-get install mysql-client(如果安装mysql-server时，mysql-client也安装了可以跳过)

3.检查安装情况

数据目录:/var/lib/mysql
配置文件:/usr/share/mysql/, /etc/mysql/
相关命令:/usr/bin/mysql*, /usr/sbin/*
启动脚本:/etc/init.d/mysql
启动服务:service mysql start
关闭服务:service mysql start
查看服务:service mysql start

4.连接数据库

mysql -h 127.0.0.1 -P 3306 -uroot -p123
